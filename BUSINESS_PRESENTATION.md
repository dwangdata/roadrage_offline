## Appendix: Technical Deep Dive
*Available for technical questions*

### Technology Stack:
- **Frontend:** Progressive Web App (PWA) - HTML5, JavaScript, CSS3
- **Sensors:** Device Motion API, Geolocation API, MediaRecorder API
- **Storage:** IndexedDB for offline data persistence
- **Mapping:** Leaflet.js with OpenStreetMap integration
- **Analytics:** Chart.js for real-time data visualization
- **Cross-platform:** Works on iOS, Android, and desktop browsers

### Key Algorithms:
- **Roughness Detection:** High-pass filter on accelerometer Z-axis data
- **Data Processing:** Variance calculation on vibration buffer
- **Geospatial Indexing:** Grid-based location discretization
- **Real-time Visualization:** Live updating map overlays with color coding

### Data Privacy & Security:
- All data stored locally by default
- Opt-in data sharing with explicit consent
- GDPR and privacy regulation compliant
- Open source architecture for transparency
