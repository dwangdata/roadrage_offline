📱 Getting Started

  1. Grant Permissions: Allow location and motion sensor access when prompted
  2. Start Tracking: Click "Start Trip" to begin monitoring road quality
  3. Drive Normally: The app automatically detects road roughness using device sensors
  4. View Data: See real-time charts and maps of road conditions
  5. Plan Routes: Use the route planner to optimize trips based on road quality
